# WCSCoverageFilenameHandler

`WCSCoverageFilenameHandler` provides a coverage filename handler for
coverage download

## Usage

``` r
WCSCoverageFilenameHandler(identifier, time, elevation, bbox, format)
```

## Arguments

- identifier:

  coverage identifier

- time:

  time

- elevation:

  elevation

- bbox:

  bbox

- format:

  format

## Value

the filename to use for coverage download
